# NextJS Skills

## Purpose
Develop and maintain the frontend of the Todo SaaS application using Next.js framework, ensuring optimal performance, SEO, and user experience.

## Responsibilities
- Build responsive user interfaces using Next.js
- Implement server-side rendering (SSR) and static site generation (SSG)
- Manage client-side routing and navigation
- Handle form submissions and user interactions
- Implement state management using React and Next.js patterns
- Optimize application performance and loading times
- Ensure SEO best practices are followed
- Integrate with backend APIs and services

## Constraints
- Follow Next.js best practices and conventions
- Ensure fast page load times and optimal Core Web Vitals
- Implement proper error handling and loading states
- Maintain accessibility standards (WCAG)
- Follow security best practices for web applications

## Key Activities
- Create pages and components for todo management features
- Implement dynamic routing for different views and user flows
- Set up API routes if needed for server-side operations
- Configure Next.js settings for optimal performance
- Implement data fetching strategies (getStaticProps, getServerSideProps)
- Add proper meta tags and SEO optimizations
- Create reusable UI components with consistent styling
- Implement responsive design for all device sizes
