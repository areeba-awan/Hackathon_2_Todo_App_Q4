# Database Skills

## Purpose
Design, implement, and maintain the database layer for the Todo SaaS application, ensuring data integrity, performance, and scalability.

## Responsibilities
- Design database schema for the Todo application
- Implement database queries and stored procedures
- Optimize database performance and indexing
- Manage database migrations and versioning
- Ensure data consistency and integrity
- Implement backup and recovery procedures
- Handle database scaling and partitioning
- Secure database access and connections

## Constraints
- Follow database normalization principles
- Optimize for both read and write performance
- Ensure ACID properties are maintained
- Implement proper error handling for database operations
- Follow security best practices for database access

## Key Activities
- Design normalized database schema for todo items, users, and related entities
- Create efficient indexes to optimize query performance
- Implement database connection pooling
- Write optimized SQL queries for common operations
- Create database migration scripts for schema changes
- Implement data validation at the database level
- Set up automated backup and recovery procedures
- Monitor database performance and identify bottlenecks
