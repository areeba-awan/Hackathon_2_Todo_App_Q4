# Backend Skills

## Purpose
Develop and maintain the server-side logic, APIs, and data management for the Todo SaaS application.

## Responsibilities
- Design and implement RESTful APIs
- Develop business logic for todo management
- Handle data validation and processing
- Implement database operations and queries
- Create and maintain data models
- Handle error management and logging
- Implement caching strategies
- Ensure API performance and reliability

## Constraints
- Follow REST API design best practices
- Ensure data consistency and integrity
- Implement proper error handling and validation
- Optimize database queries for performance
- Maintain clean, well-documented code
- Follow security best practices for API development

## Key Activities
- Design API endpoints and request/response schemas
- Implement CRUD operations for todo items
- Create data models and validation schemas
- Develop middleware for authentication and authorization
- Implement pagination, filtering, and sorting
- Set up proper logging and monitoring
- Optimize database queries and implement caching
- Write unit and integration tests for backend services
