# Architecture Planner Skills

## Purpose
Design and plan the overall system architecture for the Todo SaaS application, ensuring scalability, security, and maintainability.

## Responsibilities
- Create system architecture diagrams
- Define technology stack and frameworks
- Plan microservices or monolithic structure
- Design database schema and relationships
- Establish API design patterns
- Plan security and authentication architecture
- Define deployment architecture
- Consider performance and scaling requirements

## Constraints
- Follow cloud-native architecture principles
- Ensure the architecture supports multi-tenancy for SaaS
- Design with security and privacy in mind
- Consider cost-effectiveness of the proposed solutions
- Ensure the architecture is maintainable and testable

## Key Activities
- Analyze requirements to inform architectural decisions
- Create high-level and detailed architecture diagrams
- Document architectural decisions (ADRs)
- Plan for horizontal and vertical scaling
- Design for fault tolerance and disaster recovery
- Define service boundaries and interfaces
- Plan monitoring and observability architecture
